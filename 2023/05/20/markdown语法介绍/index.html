什么是Markdown语法？

Markdown 语法是轻量级文本标记语言，Markdown简单易学，功能强大。它的思想是”关注写作本身，沉浸式写作“。通俗的讲，写作的时候我连鼠标我都不要动它，谁能做到？markdown可以。

Markdown语法支持什么修饰？

  • 标题 （标题1 ~ 标题6）
  • 图片（本地图片、远程图片）
  • 链接
  • 强调 ( 加粗、倾斜 )
  • 修饰线 （分割线、 中划线）
  • 引用 （单条引用、多条引用）
  • 代码块
  • 上标下标
  • 列表 （无序列表、有序列表）
  • 表格

8. 上标下标

上标和小标经常在数学或者化学中应用到，比如说 三角函数公式，用到上标，水分子用到了下标。
